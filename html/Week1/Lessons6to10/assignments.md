# [assignment2](./assignment2.html)

* Is there a difference between these elements when they are shown on the page or not?

```html
<p class="element">Welcome To The New World</p>
<p class='element'>Welcome To The New World</p>
<p class=element>Welcome To The New World</p>
```

> **Answer ==> There is no difference**
